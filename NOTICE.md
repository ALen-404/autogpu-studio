## 本仓库改动方向

本仓库当前围绕以下方向进行非商业开源设计与后续开发：

- 用户自带 AutoDL 账号或实例。
- AutoDL 官方入口和实例连接配置。
- 用户自部署环境中的 AutoDL API Key 自动化。
- 远程 GPU Worker。
- 本地视频、图片和 TTS 模型目录。
- 多模态生成任务路由。

本仓库不提供商业租机平台能力，不做 AutoDL 算力转售，不在 AutoDL 价格上加价，不做站内余额充值、冻结、扣费或抽成。AutoDL 相关费用由用户与 AutoDL 官方自行结算。
